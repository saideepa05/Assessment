# Assessment Dataset

## Files included

- `bu_financials_q3.csv` — Quarterly financial data for three business units across Q1–Q3 2024. Contains revenue, EBITDA, and headcount actuals vs. plan.
- `commentary_excerpts.txt` — Five internal document excerpts. Treat each `--- DOCUMENT N ---` block as a separate source in your corpus.

## Notes on the data

- Not all documents are equally relevant to every question. Some contain overlapping information; some contain gaps.
- Document 5 explicitly notes that it is a summary and does not contain complete variance detail — this is intentional.
- Headcount figures in the CSV are end-of-quarter actuals.
- All financials are in USD millions unless noted otherwise.

## What "source citation" means for this exercise

For any generated answer, we'd like to see which specific document(s) — and ideally which section or passage — the answer is grounded in. This doesn't need to be production-grade; it just needs to be traceable.
